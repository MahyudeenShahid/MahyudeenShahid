# 🛠️ How to Use This Profile README Template

You are welcome to clone, copy, and modify this repository to create your own premium GitHub Profile README!

## 🚀 Steps to Setup

### 1. Create a Template Copy
* Click the green **"Use this template"** button at the top of this repository page to create a copy in your own account.

### 2. Customize the README
* Open `README.md` and search-and-replace all instances of the username `MahyudeenShahid` with your own GitHub username.

### 3. Customize the SVG Banners
* The header banners are located in `readmefile/dark.svg` and `readmefile/light.svg`.
* Open these files and edit the text layers (like location, education, portfolio link, and email) to match your own information.

### 4. Enable GitHub Theme-Switching
* The `<picture>` tags inside the `README.md` already configure prefers-color-scheme selectors to flip between light and dark modes automatically.

### 5. Final Push
* Rename your repository to match your GitHub username exactly (e.g. if your username is `JohnDoe`, your repo must be named `JohnDoe`).
* Commit and push your changes to your default branch!
